# SOAR_EDR-project

This project demonstrates the step-by-step process of building and organizing an automated SOC workflow.

The included file SOAR_EDR.drawio (which can be viewed on draw.io) provides a detailed diagram illustrating the system's workflow and the interaction between components.

In this project, I utilized LimaCharlie as the EDR (Endpoint Detection and Response) to detect potential threats and system compromises. Additionally, Tines acts as the SOAR (Security Orchestration, Automation, and Response) platform to orchestrate the communication between different security tools and automate the response process
